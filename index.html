Método Kanban

Todo lo que debes conocer de Kanban

La metodologia que debes usar

¿Quiere entender qué es Kanban? Descubre todo el potencial del método Kanban para gestionar el trabajo y aprenda a aplicarlo con éxito en su organización.

Descripción

Kanban hace referencia a tarjetas visuales.

-Para esta metodologia se debe crear un tablero en el cual deben participar todos los miembros del equipo

-Como es un metodo visual,es mas facil ver en que estado esta el proyecto

-Se asignan las tareas de manera mas efectiva

Ventajas

Transparencia
Evita tareas ineficientes
Control de tareas
Flexibilidad

Características

Visualización de todas las tareas
En proceso
Priorización según importancia y urgencia
Seguimiento del tiempo

Conclusión

Esta metodologia es un gran aporte para la ingeniería de software, ya que comos los otros metodos nos aporta otra variente para el desarrollo , ya que cada equipo y persona no tiene un solo metodo , si no que puede elegir con el que se sienta mas comodo y considero que este metodo es didactico que como puede tener ventajas , puede tener bastantes desventajas pero esta en eso donde uno como desarrollador escoge que metodo es mejor para usar.
